Hi visitors!
This project is a bookstore Windows Form for managing books and users in it.
In this program you are able to:

1. Add admin or normal users.
2. Remove admin or normal users.
3. Add books.
4. Remove books.
5. Edit books information.
6. Search books.
7. Search Authors.
8. Borrow books.

This project is for educational purposes and is not a professtional program to be considered.
